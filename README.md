# 🚀 Sistema de Gerenciamento de Funcionários

Este projeto foi desenvolvido como parte de um teste técnico, com o objetivo de aplicar conceitos fundamentais de Java, como orientação a objetos, manipulação de coleções e boas práticas de desenvolvimento.

---

## 🧠 Funcionalidades

O sistema realiza as seguintes operações:

* Cadastro de funcionários a partir de uma estrutura pré-definida
* Remoção de funcionário por nome
* Impressão formatada dos dados (data e salário)
* Aplicação de aumento salarial
* Agrupamento de funcionários por função
* Filtro de funcionários por mês de aniversário
* Identificação do funcionário mais velho
* Ordenação alfabética dos funcionários
* Cálculo do total de salários
* Cálculo de quantos salários mínimos cada funcionário recebe

---

## 🧱 Estrutura do Projeto

O projeto foi organizado seguindo boas práticas de separação de responsabilidades:

```
src/main/java/org/example
├── model       # Classes de domínio (Pessoa, Funcionario)
├── service     # Regras de negócio
├── util        # Utilitários (formatação)
└── Principal   # Classe principal (execução)
```

---

## 🛠️ Tecnologias e conceitos utilizados

* Java 24
* Programação Orientada a Objetos (POO)
* Stream API
* Collections (List, Map)
* BigDecimal (precisão em valores monetários)
* DateTime API (LocalDate, Period)
